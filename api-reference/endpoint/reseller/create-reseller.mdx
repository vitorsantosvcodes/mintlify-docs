## Como utilizar este endpoint?

Esse endpoint permite criar um novo revendedor, definindo suas permissões e dados de acesso.

### Headers obrigatórios

<ParamField
  header="Authorization"
  type="string"
  placeholder="Bearer {token}"
  required
>
  O token de autenticação no formato Bearer é obrigatório para acessar este endpoint.
</ParamField>

### Parâmetros do corpo da requisição (Payload)

<ParamField body="username" type="string" placeholder="" required>
  Nome de usuário do revendedor.
</ParamField>

<ParamField body="password" type="string" placeholder="" required>
  Senha do revendedor.
</ParamField>

<ParamField body="is_master" type="number" placeholder="1" required>
  Define se o revendedor será master (**1**) ou não (**0**).
</ParamField>

<ParamField body="credits" type="string" placeholder="12312" required>
  Quantidade de créditos disponíveis para o revendedor.
</ParamField>

<ParamField body="whatsapp" type="string" placeholder="12312312312">
  Número de WhatsApp do revendedor. Campo opcional.
</ParamField>

<ParamField body="telegram" type="string" placeholder="zeca">
  Usuário do Telegram do revendedor. Campo opcional.
</ParamField>

<ParamField body="language" type="string" placeholder="pt">
  Idioma do revendedor (ex.: **pt**, **en**).
</ParamField>

<ParamField body="country" type="string" placeholder="Brasil" required>
  País de origem do revendedor.
</ParamField>

<ParamField body="email" type="string" placeholder="teste@gmail.com">
  E-mail do revendedor. Campo opcional.
</ParamField>

<ParamField body="notes" type="string" placeholder="Observações">
  Observações adicionais. Campo opcional.
</ParamField>

### Exemplo de Requisição

```json
POST https://mcapi.knewcms.com:2087/users
Headers:
{
  "Authorization": "Bearer seu_token_aqui",
  "Content-Type": "application/json"
}
Payload:
{
  "username": "username-definido",
  "password": "password-definido",
  "is_master": 1,
  "credits": "12312",
  "whatsapp": "12312312312",
  "telegram": "zeca",
  "language": "pt",
  "country": "Brasil",
  "email": "teste@gmail.com",
  "notes": ""
}
```

</ResponseField>

  Senha do revendedor.
